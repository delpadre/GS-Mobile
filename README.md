# SkillUpPlus 2030+

## Descrição do Projeto

O SkillUpPlus 2030+ é um aplicativo mobile para requalificação profissional, desenvolvido com React Native e TypeScript. A plataforma oferece trilhas de aprendizado em tecnologias emergentes, sustentabilidade e habilidades do futuro, alinhada com os Objetivos de Desenvolvimento Sustentável da ONU.

## Integrante

## Funcionalidades

- Sistema de autenticação e validação
- Dashboard com métricas pessoais
- Catálogo de cursos com filtros
- Acompanhamento de progresso
- Perfil personalizável
- Navegação híbrida (Stack, Drawer, Tabs)

## Tecnologias

- React Native
- TypeScript
- Expo
- React Navigation

## ODS Conectados

- ODS 4 - Educação de Qualidade
- ODS 8 - Trabalho Decente e Crescimento Econômico
- ODS 9 - Indústria, Inovação e Infraestrutura
- ODS 10 - Redução das Desigualdades
